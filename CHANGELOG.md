# Changelog

All notable changes to this module will be documented in this file.

## [0.1.1] - 2026-04-09

Initial public MVP release.

### Added

- Configurable secondary AoE activity for `D&D5e` items.
- Support for complete `Midi-QOL` workflows, including player-side workflows when a full Midi workflow is produced.
- Dedicated Secondary AoE configuration window.
- Header button on the item sheet to open the Secondary AoE configuration.
- French and English localization support.
- Debug mode for module diagnostics in the Foundry console.

### Notes

- Automatic triggering is designed for complete `Midi-QOL` workflows rather than chat-card-only usage.
- This MVP keeps the current AoE engine, triggers, functional UI flow, GM/PJ support, and socket relay architecture intact.
